## Introduction

Welcome to Level 1 — where every great journey begins! In this level, you'll learn the **fundamental building blocks** of Commodore 64 BASIC programming. The C64 was the home computer that brought games and creativity to millions of kids in the 1980s, and BASIC was how they made magic happen.

You will learn how to **store information, do math, get player input, make decisions, repeat actions, and organize all of it together** to build a classic number-guessing game. Every computer game and app was built from these same building blocks.

## The Big Picture

Every C64 program follows this simple pattern:

```
10 Get input from the player
20 Do math or make decisions
30 Show results on screen
40 Repeat (or end)
```

You already know how to type BASIC commands into the C64. In this level, you'll learn how to **chain them together** to create programs that are bigger than any single line.

### Level 1 at a Glance

- **Variables & Types** → How C64 BASIC stores information (numbers, text, and the magic `=` sign)
- **Expressions** → How to do arithmetic math, control what order it happens, and pick random numbers
- **INPUT & PRINT** → How commands work together to let you talk to the computer
- **IF / THEN / GOTO** → How to make your program make decisions — "IF this, THEN that"
- **FOR / NEXT** → How to make your program repeat something over and over (loops!)
- **Arrays (DIM)** → How to create lists and tables inside your program's memory
- **Strings** → How to work with text: counting characters, cutting and pasting text
- **GOSUB / RETURN** → How to write reusable code blocks (like mini-programs inside your program)
- **Capstone: Guess the Number** → Put everything together into a fun game!

## Style Guide for This Level

- All C64 BASIC commands use **uppercase** (as they were typed on the original C64)
- Line numbers are always multiples of **10** for easy editing
- The `=` sign is used for **both assignment and comparison**
- String variables end with `$` (like `NAME$`)
- Numeric variables have no suffix (called `SCORE`, not `SCORE$`)
- `CHR$(147)` is the code to clear the C64 screen
- The screen is **40 characters wide by 25 lines**
