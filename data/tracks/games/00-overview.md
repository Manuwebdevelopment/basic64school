# Games Track — OVERVIEW

## Track Prerequisites
Before entering this track, complete:
- Level 0 through Level 2
- Level 6 (at least the state machine and scoring lessons)

## What You'll Build
Five lessons that take you from a blank screen to a complete, polished C64 game with levels, collisions, and a full game loop.

## Track Structure
1. **Game Prototypes** — Build a minimal playable game in under 10 lines of BASIC
2. **Asset Packing** — Encode sprites, maps, and tile sets as DATA blocks for maximum efficiency
3. **Collision Detection** — Reliable hit detection for sprites and grid-based objects
4. **Level Design** — Multiple levels with DATA-driven loading and a map parser
5. **Polish & Playtest** — SFX, transition effects, difficulty tuning, playtesting strategies

## Game of the Track
By lesson 5, you will have built **"STAR CHASER" — a complete C64 game with multiple levels, sprites, scrolling, scoring, and difficulty.**
