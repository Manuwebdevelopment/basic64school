# Creative Coding Track — OVERVIEW

## Track Prerequisites
Before entering this track, complete:
- Level 2 (Screen & PETSCII)
- Level 5 (Data Structures & Effects)

## What You'll Build
Four lessons that take you from simple PETSCII text art to full demoscene-style animated effects using only C64 BASIC and screen memory.

## Track Structure
1. **Text Art** — Drawing art using only PRINT characters and PETSCII codes
2. **Generative Patterns** -- Creating art with RND() for fractals, kaleidoscopes, and random landscapes
3. **Demoscene Techniques** -- Scrolling graphics, color cycling, animated backgrounds
4. **Performance Optimization** -- Making text effects run smoothly on a 1982 machine

## Art of the Track
By lesson 3, you will have created a C64 demoscene-style demo with animated graphics, scrolling text, and color cycling.
