# Systems & Tools Track — OVERVIEW

## Track Prerequisites
Before entering this track, complete:
- Level 1 (BASIC Basics, Arrays/DIM)
- Level 5 (Data structures, file I/O concepts)

## What You'll Build
Four lessons that teach you to build useful C64 BASIC tools: a text editor, a sprite editor, a disk file system, and a multi-program menu system.

## Track Structure
1. **Basic Text Editor** — Build a functional text-based editor using BASIC arrays, GET, and screen management
2. **Sprite Editor** — A C64 BASIC program that lets you draw sprites with the keyboard, then save them
3. **Disk File I/O System** — Save and load programs, data files, and settings using C64 BASIC's disk system
4. **Multi-Program Menu System** — A menu-driven multi-part application with nested menus and save states

## Tool of the Track
A complete C64 BASIC toolkit: a text editor, a sprite designer, and a file manager — all built in BASIC and running on the C64 emulator.
